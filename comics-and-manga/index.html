  • NovelPlanet - Novelas gratuitas e inglesas.
  • manga_downloader - Scraper de Python que descargará mangas de mangareader.net.
  • The Free Manga Downloader (FMD) - Aplicación gratuita de código abierto escrita en Object Pascal para administrar y descargar manga desde varios sitios web.
  • Tachiyomi - Lector de manga gratuito y de código abierto para Android.
  • MangaZone - Una aplicación de lectura de manga.
  • Kindle Comic Converter - Conversor de cómics y manga para lectores de libros electrónicos.
  • readcomiconline.to - Manga y cómics subidos a diario.
  • Comic Extra - Cargas diarias de cómics, interfaz de usuario limpia.
  • GetComics - Comenzó como una alternativa para obtener cómics, particularmente cómics de DC y Marvel basados en Estados Unidos.
    • Omnibus - Busca y descarga cómics que se agregan a GetComics.info fácilmente.
  • Gazee! - descontinuado Un lector de cómics webApp para tus cómics digitales favoritos.
  • Comix-Load - Enlaces de descarga directa para cómics y manga en inglés y alemán.
  • MangaDex - MangaDex es un lector de manga en línea que se adapta a todos los idiomas.
  • KissManga - Otro sitio web de manga.
  • r / manga - ¡Todo y cualquier cosa en mangas! (¡Manhwa también!).
  • Madokami - Requiere registro (actualmente cerrado), consulta los espejos de archive.org a continuación.
    • Madokami 0-E - Descarga títulos de manga con el nombre 0 a E.
    • Madokami F-K - Descarga títulos de manga llamados F a K.
    • Madokami L-Q - Descarga títulos de manga llamados L a Q.
    • Madokami R-Z - Descarga títulos de manga llamados R to Z.
    • Novelas, brutos y libros de arte de Madokami - Descarga novelas, bocetos de manga y libros de arte.
  • NineAnime - Sitio de manga actualizado / activo.
  • HakuNeko - Un descargador multiplataforma de manga y anime de +400 sitios web. Manga y anime en múltiples idiomas y formatos (cbz, carpeta con imágenes, epub).
  • ComicPunch.net - Anteriormente ReadComicsOnline.net.
  • ReadComicsOnline - Versión rusa del antiguo ReadComicsOnline.
  • Mangareader.net - Lee tus escaneos de manga favoritos en línea.
  • Mangadoom - ¡Leer manga gratis en línea !.
  • Manga Park - Alternativa a KissManga.
  • Mangago - ¡Lee manga online gratis!
  • Readmng - ¡Leer manga gratis en línea!
  • Manga Life - ¡Leer manga gratis en línea!
  • Mangaseeonline - Normalmente carga escaneos oficiales.
  • Mangabat - Leer Manga Online gratis.

  • The Free Manga Downloader (FMD) - Aplicación gratuita de código abierto escrita en Object Pascal para administrar y descargar manga desde varios sitios web.
  • n1mo / manga - Un descargador de mangareader.net de código abierto escrito en Python.
  • mangadex-dl - Descarga manga de MangaDex.org.
  • HakuNeko - Descargador de manga y anime para Linux, Windows y macOS.
