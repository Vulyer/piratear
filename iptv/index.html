iptv

  • iptv-org/iptv - Colección de más de 8000 canales de IPTV disponibles públicamente de todo el mundo.
  • fastocloud - Panel de administración de IPTV/Video en la nube para servidores.
  • Reproductor de IPTV de TiviMate - Una popular aplicación de Android para ver IPTV en decodificadores de Android.
  • Códigos Xtream - La aplicación Smarters permite a los usuarios de Android utilizar IPTV, ofrece contenido gratuito y de pago.
  • r/IPTV - Subreddit puede resultar útil para medir el estado actual de los proveedores de IPTV.
  • r/iptvresellers - Promociones y anuncios de proveedores de IPTV.
  • r/IPTVReviews - Reseñas de proveedores de servicios de IPTV.
  • Proyectores de DVD de los Oscar 2019 - Lista de proyectores de DVD para los Oscar de 2019.
  • Academy Awards 2019 Screeners Megathread - Publicación de/u/idoideas que enumera todos los lanzamientos de DVDSCR disponibles para 2019.
  • allsprk.tv - Sitio de transmisión en vivo por salto de canal con una sala de chat.
  • antennas - Emulador HDHomeRun para Plex DVR para conectarse a Tvheadend.
  • Comunidad de IPTV - Sitio web de debate sobre tecnología e IPTV, útil para encontrar un proveedores/revendedores de IPTV.
  • Lista de proveedores de IPTV - Colección de más de 8000 canales de IPTV disponibles públicamente de todo el mundo.
  • MythTV - Grabador de video digital de software de código abierto gratuito.
  • PIPTV - Cliente Python IPTV con más de 80 canales.
  • Piratería y automatización, una guía general de /u/JukeCity101
  • STBEmulator - Aplicación popular de Android para usar transmisiones de IPTV con EPG.
  • tele - Proxy de IPTV para Plex Live escrito en Golang.
  • tvheadend - Tvheadend es un servidor de transmisión de TV para Linux compatible con DVB-S, DVB-S2, DVB-C, DVB-T, ATSC, IPTV, SAT> IP y otros formatos a través de la tubería Unix como fuentes de entrada.
  • UlstreaMix - Sitio de transmisión de TV en vivo, predominantemente deportes.
  • xTeVe - Proxy M3U para Plex DVR.
  • Xtream Editor - Xtream Editor permite crear, editar y ordenar listas de reproducción m3u en línea.
  • Guía y conceptos básicos de IPTV - Publicación de/u/Specific_Angle que describe en detalle cómo comenzar con IPTV.

Acestreams#

  • aceproxy - Proxy HTTP Ace Stream. (abandonware).
  • acestream.org - Ace Stream es una aplicación de transmisión entre pares que permite transmitir deportes en vivo y otros contenidos.
  • AceStreamSearch - Busqueda de transmisiones de Ace.
  • iktason/aceproxy - Una imagen de Docker para ejecutar aceengine + aceproxy, p. ej. para ver Torrent-TV.ru.
