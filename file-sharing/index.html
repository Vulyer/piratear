Uso compartido de archivos

  • Cargador de archivos e imágenes - El programa más completo para cargar archivos e imágenes en diferentes servidores.
  • CrossFTP - CrossFTP hace que sea extremadamente sencillo administrar las tareas relacionadas con FTP.
  • D-lan - Un software gratuito para compartir archivos en LAN.
  • DBREE - DBREE es una forma sencilla y sencilla de cargar y compartir cualquier tipo de archivo.
  • Envelop - Aplicación para compartir archivos descentralizada a través de Web / [Android](https://play.google.com /store/apps/details?id=app.envelop).
  • FilePizza - Transferencias gratuitas de archivos punto a punto en su navegador.
  • FileZilla - La solución FTP gratuita y de código abierto (S).
  • Flareget - Acelerador y administrador de descargas multiproceso con todas las funciones.
  • Fopnu - Fopnu es una herramienta de red nueva y descentralizada para compartir archivos a través de P2P.
  • Nord Locker - Almacena y comparte tus archivos de forma segura, 5 GB son gratis.
  • Sitio web de Mega.nz - Fácil sincronización automatizada entre tus computadoras y tu unidad en la nube MEGA.
  • Meiga - Una herramienta que permite compartir directorios locales seleccionados a través de la web.
  • NitroShare - Aplicación de transferencia de archivos de red multiplataforma.
  • OnionShare - Comparta de forma segura y anónima un archivo de cualquier tamaño.
  • Magnetissimo - Aplicación web que indiza todos los sitios populares de torrents y los guarda en la base de datos local.
  • Muwire - Programa de intercambio de archivos basado en I2P.
  • Plowshare - Herramienta de línea de comandos y motor para administrar sitios web para compartir.
  • PushBullet para escritorio - La aplicación de escritorio que faltaba para Pushbullet.
  • PushBullet - Pushbullet conecta tus dispositivos, haciéndolos sentir como uno.
  • Quazaa - Un cliente de intercambio de archivos (P2P) multiplataforma y de múltiples redes.
  • Wormhole - Obtén archivos y directorios de tamaño arbitrario (o fragmentos de texto cortos) de una computadora a otra de forma segura.
  • aria2 - Utilidad de descarga de línea de comandos multiprotocolo y fuente múltiple liviana.
  • uds - Utilidad de código abierto para dividir sus archivos en partes babse64 que son lo suficientemente pequeñas como para cargarlas en Google Docs.
