irc

Internet Relay Chat (IRC) es un protocolo de chat abierto. Cualquiera puede poner o conectarse a un servidor y chatear sin necesidad de crear cuentas o registrarse.

  • autodl-irssi - Un complemento para irssi que monitorea los canales de anuncios de IRC para rastreadores de torrents y descarga archivos torrent basados en filtros definidos por el usuario.
  • Tutorial de XDCC - Descarga de XDCC para principiantes: ¡Hazlo como un profesional!
  • XDCC - XDCC (Xabi DCC o eXtended DCC) - Método de intercambio de archivos de computadora que utiliza Internet Relay Chat (IRC) red como servicio de host.
  • ZNC - Un saltador (bouncer) de IRC avanzado.

Clientes#

  • irssi - Su aplicación de chat en modo texto desde 1999.
    • autodl-irssi - Un complemento para irssi que monitorea los canales de anuncios de IRC para rastreadores de torrents y descarga archivos torrent basados en filtros definidos por el usuario.
    • Notificaciones de Slack para autodl-irssi - Guía tuya sobre cómo habilitar las notificaciones de Slack para autodl-irssi.
    • autodl-curl-sonarr - Secuencia de comandos para usar como comando de carga para que autodl-irssi publique en Sonarr.
    • mreg - Genera una expresión "Coincidir lanzamientos" para su filtro autodl-irssi basada en la sección "Fechas de lanzamiento de DVD más solicitadas" de dvdsreleasedates.com.
  • mIRC - El cliente IRC original para Windows.
  • HexChat - HexChat es un cliente de IRC basado en XChat, pero a diferencia de XChat, es completamente gratuito tanto para Windows como para sistemas similares a Unix.
  • Kiwi IRC - Cliente de IRC web receptivo con soporte para temas.
  • KVIrc - Otro cliente gráfico de IRC.
  • Shout - El cliente de IRC web autohospedado.
  • The Lounge - TheLounge (una bifurcación de shoutIRC) - es un cliente de IRC web que alojas en tu propio servidor.
  • weechat - El cliente de chat extensible.
  • Convos - Cliente web IRC siempre en línea.
  • InspIRCd - Servidor IRC modular escrito en C++ para Linux, BSD, Windows y macOS.
  • Dispatch - Cliente de IRC web autohospedado escrito en Go.
  • ngircd - Servidor de Internet Relay Chat gratuito, portátil y ligero para redes pequeñas o privadas.

Motores de búsqueda de IRC#

  • ixIRC - Permite buscar a través de 17 redes IRC, 32 canales y más de 189915 paquetes XDCC suministrados por el usuario.
  • SunXDCC - Otro motor de búsqueda de archivos XDCC.
  • xdcc.eu - El motor de búsqueda XDCC indexa paquetes de una gran cantidad de redes.
  • xWeasel - xWeasel es un cliente de descarga independiente y gratuito basado en IRC.
