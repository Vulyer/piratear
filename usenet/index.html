usenet

  • Usenet - Usenet es un sistema de discusión distribuido en todo el mundo disponible en computadoras.
  • Grupo de noticias de Usenet - Un grupo de noticias de Usenet es un repositorio generalmente dentro del sistema de Usenet, para los mensajes enviados por muchos usuarios en diferentes ubicaciones usando Internet.
  • r/Usenet - Una comunidad próspera dedicada a ayudar a los usuarios nuevos y antiguos a comprender y usar Usenet.
  • Introducción a usenet -/r/usenet wiki
  • r/UsenetInvites - Solicitudes y ofertas para indexadores de Usenet.
  • NZBLINK - El esquema de URI NZBLNK™ define el formato de los enlaces NZBLNK™ para identificar el contenido binario de Usenet y proporciona información adicional para manejar ese contenido correctamente (similar a los enlaces magnéticos, pero para NZB).
  • Usenet-Uploaders - Tabla de aplicaciones para cargar contenido en Usenet.
  • QuickPar - Herramienta para reconstruir archivos binarios de Usenet dañados/faltantes/corruptos.
  • NZB Monkey - Herramienta de ayuda de descarga de NZB.
  • SABconnect ++ - Extensión de Chrome que agrega botones 'Enviar a SABnzbd' con un solo clic a muchos sitios populares de índices NZB.
  • ngPost - Línea de comandos (o GUI minimalista) - Póster de usenet para binarios desarrollados en C ++/QT.
  • Una guía rápida para elegir un proveedor de Usenet - Publicación en Reddit de/u/FlickFreak.

Indexadores#

  • r/Usenet wiki: indexadores - Información sobre los servicios de indexación favoritos de/r/Usenet.

Indexadores pagos#

  • DOGnzb - Sitio de NZB solo para invitados (aunque tienen una página de registro que ocasionalmente se abre durante el mantenimiento).
  • DrunkenSlug - Indexador NZB popular con un nivel gratuito y una retención decente.
  • NZBCat - NZBCat es un indexador nZEDb NZB solo por invitación.
  • NZBFinder - Indexador de Usenet y API newznab con una interfaz de usuario limpia y más de 8 años de acumulación de NZB.
  • NZBgeek - Indexador de Usenet asequible que funciona desde 2014.
  • omgwtfnzbs - Indexador NZB solo para invitados con un nombre divertido.

Indexadores gratuitos#

  • BINZB - Indexador minimalista.
  • Binsearch - Con este sitio puede buscar y explorar grupos de noticias binarios de Usenet.
  • FindNZB - Un antiguo indixer, los resultados están desactualizados y, a menudo, sin conexión.
  • NZBIndex - El primer indexador de Usenet gratuito que encontrará en los resultados de búsqueda de Google.
  • NZBFriends - Uno de los indexadores más antiguos, lamentablemente los resultados también suelen estar desactualizados.
  • NZBKing - El servicio le permite buscar y explorar archivos binarios que se han publicado en los grupos de noticias de Usenet.
  • Usenet Crawler - Indexador de Usenet con acceso API para usuarios registrados.
  • NewzLeech - Newzleech es un motor de búsqueda de archivos de Usenet.
  • GingaDADDY - Otro indexador gratuito popular de NZB, requiere registrarse.

Software de indexación#

  • Momentum Plus - Usenet-Newsreader y en el (futuro cercano) - indexador y recopilador de archivos de Usenet.
  • nZEDb-deploy - Una colección de scripts para automatizar y simplificar la implementación de un indexador de Usenet nZEDb usando el nuevo formato de su repositorio de GitHub.
  • nZEDb - Una bifurcación de nnplus (2011) - | Indexador NNTP/Usenet/Newsgroup.
  • newznab-tmux - Indexador usenet basado en Laravel.
  • newznab - Newznab es una aplicación de indexación de Usenet, que facilita la creación de una comunidad de Usenet.
  • NzBlite - (nzblite client) - Formato de archivo de código abierto para simplificar la carga y descarga de Usenet en grupos de noticias.

Clientes de Usenet#

  • NZBget - Descargador eficiente de Usenet escrito en C++.
  • NZBVortex 3 - Simplemente el mejor cliente de Usenet para Mac.
  • alt.binz - Un poderoso lector de noticias binario, para descargar y administrar artículos de Usenet.
  • Newsbin - Newsbin es un software para los sistemas operativos Microsoft Windows que descarga archivos de los grupos de noticias de Usenet.
  • NZBGet - Eficiente descargador de Usenet escrito en C++ que es un poco más liviano en recursos del sistema que SABNZBd.
  • PowerNZB - El último cliente binario de Usenet para Android.
  • SABnzbd - SABnzbd es un lector de noticias binario de código abierto escrito en Python.
  • spotweb - Spotweb es una comunidad Usenet descentralizada basada en el protocolo Spotnet.
  • Unison - Aplicación macOS para acceder a los grupos de noticias de Usenet y las muchas maravillas y misterios que contiene (descontinuado).
  • Usenetic - El cliente de Usenet con todas las funciones para macOS.
